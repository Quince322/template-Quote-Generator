View

It manages to make the DOM change in predictable ways that are easy to scale. Can be used with mobile, VR, and desktop apps. Anything with a view. You can build individual components, save them, then never have to write them again. They are saved and you can share them with others. Data always flows from parent to child: flows downstream.

Virtual DOM

Always minimize DOM manipulation to not make web browsers work so hard. Increases performances and decreases bugs. Instead of giving commands to the DOM we use React to make changes in the most optimal way possible.
